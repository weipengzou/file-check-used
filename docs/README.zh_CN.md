一个检测项目中未使用的代码工具。

支持

- 资源素材
- 常量
- TS 类型声明（支持自动删除）

方便去除未使用素材，降低项目打包的大小。

毫秒级速度，项目零侵入，大小仅 3KB。

## 安装

```shell
npm i -g file-check-used
```

## 用法

全局安装后在项目根目录中使用 `fcu`命令。

```shell
$ fcu
```
